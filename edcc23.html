    Abstract: Modern Internet of Things (IoT) architectures encompass a wide array of computational logic and communication protocols. However, the security vulnerabilities inherent in these systems pose significant risks. It is imperative to address these threats in designing communication protocols to ensure robust and secure communication system. In this study, we propose an approach that utilizes the Concurrent Stochastic Game model to precisely specify the behavior of IoT communication entities, referred to as players, while considering potential data corruption attacks at the communication edge. We implement the RabbitMQ communication protocol in the PRISM-games language for automated analysis, leveraging reward Probabilistic Alternating Temporal Logic (rPATL) to model functional and security requirements as game goals. Through this assessment, we gain valuable insights into the protocol's feasibility in relation to the existing implementation of the Sensinact edge gateway. To validate our approach, we investigate a data corruption attack targeting sensed data and examine its impact on payload routing and message loss for water dam infrastructure. .
